1.  For sum of all divisors of a number N, find its prime factorization of the form:
    N = 2^a * 3^b * 5^c ......
    sum of all divisors S = (2^0 + 2^1 + 2^2 + ...2^a)(3^0 + 3^1 + 3^3 +....3^b)(5^0 + 5^1 + 5^2 +...5^c)....
    sum of proper divisor S(p) = S - N.
